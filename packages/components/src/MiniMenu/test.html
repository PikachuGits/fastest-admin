MiniMenu Component

从编译后的HTML代码成功转换为React组件

✅ MiniMenu组件已成功创建
ℹ️ 组件特性：
  • 使用Iconify图标系统替代@mui/icons-material
  • 支持多级嵌套导航
  • 可折叠的导航分组
  • 支持禁用状态和信息标签
  • 完全的TypeScript类型支持

组件文件结构

packages/components/src/MiniMenu/
├── MiniMenu.tsx     # 主组件文件
├── demo.tsx         # 演示组件
├── index.ts         # 导出文件
└── test.html        # 测试页面

使用方法

import { MiniMenu } from '@fastest/components';

function App() {
  return (
    <MiniMenu 
      sx={{
        maxWidth: 320,
        boxShadow: 2,
      }}
    />
  );
}

主要特性

  • 导航结构：支持多级嵌套的导航菜单
  • 图标系统：使用Iconify提供丰富的图标选择
  • 交互状态：支持展开/折叠、激活状态、禁用状态
  • 信息标签：支持显示数字标签和状态信息
  • 自定义样式：通过sx属性支持完全的样式定制
